
---
title: 圆函数恒等式
taxon: table
---

|  | $\alpha + \beta$ | $\alpha - \beta$ |
| - | - | - |
| $\cos$ | $\cos\alpha \cos\beta - \sin\alpha \sin\beta$ | $\cos\alpha \cos\beta + \sin\alpha \sin\beta$ |
| $\sin$ | $\sin\alpha \cos\beta + \cos\alpha \sin\beta$ | $\sin\alpha \cos\beta - \cos\alpha \sin\beta$ |
| $\tan$ | $\dfrac{\tan\alpha + \tan\beta}{1 - \tan\alpha \tan\beta}$ | $\dfrac{\tan\alpha - \tan\beta}{1 + \tan\alpha \tan\beta}$ |
| $\cot$ | $\dfrac{\cot\alpha \cot\beta - 1}{\cot\alpha + \cot\beta}$ | $\dfrac{\cot\alpha \cot\beta + 1}{\cot\beta - \cot\alpha}$ |

|  | $\alpha + \beta$ | $\alpha - \beta$ |
| - | - | - |
| $\cosh$ | $\cosh x \cosh y + \sinh x \sinh y$ | $\cosh x \cosh y - \sinh x \sinh y$ |
| $\sinh$ | $\sinh x \cosh y + \cosh x \sinh y$ | $\sinh x \cosh y - \cosh x \sinh y$ |
| $\tan$ | $\dfrac{\tanh x + \tanh y}{1 + \tanh x \tanh y}$ |$\dfrac{\tanh x - \tanh y}{1 - \tanh x \tanh y}$ | 
| $\coth$ | $\dfrac{1 + \coth x \coth y}{\coth x + \coth y}$ | $\dfrac{1 - \coth x \coth y}{\coth x - \coth y}$ |
